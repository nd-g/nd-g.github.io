Date a Deafie

About Us

Date A Deafie is a Deaf-led project to make dating in the Deaf world more accessible. Have you ever wanted a dating app that is centered around sign language and Deaf accessibility? This project aims to develop that and will aim to create an easier Deaf dating experience!

Why multiple text modes?

These modes aim to make the website more colorblind and DeafBlind friendly. This website strives to be as DeafBlind friendly as possible as the Deaf community includes many blind folx! Any suggestions to improve DeafBlind accessibilty can be emailed directly and we will try to implement them!

Contact
